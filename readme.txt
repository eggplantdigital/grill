== Description ==
 
Grill allows you to add Projects, FAQs and Team members to your WordPress blog and provides developers with a useful framework to create your own Post Types.

Designed to work with Salt Theme and allow Nonprofits to get a website up and running easily. Use this plugin to tell the world about your great projects, team members and include frequently asked questions.

Major Features Include:

* Creates 3 different content post types in your WordPress blog
* Includes Projects, FAQs and Team Members
* Easily create different layouts by inserting the post types into a page
* Grid and list layouts available
* Includes a framework for developers to create their own post types
* Designed to work with Salt Theme, demo available here - http://demo.charitythemes.org/salt

== Installation ==
 
This section describes how to install the plugin and get it working.
 
1. Upload the Grill plugin to your blog
1. Activate it
1. Go to Settings > Grill and turn on the post types you want to use
